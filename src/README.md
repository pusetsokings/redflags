# SafeSignals - Red Flags Tracker

A comprehensive, privacy-first web application designed to help users identify and track warning signs in relationships, workplace dynamics, and family interactions. Built with React, TypeScript, and advanced AI-powered pattern recognition.

## üåü Features

### Core Functionality

#### 1. **Secure Journaling**
- Private, encrypted journal entries
- Mood tracking (1-5 scale)
- Emotion tagging with visual indicators
- Context-aware entries (romantic, workplace, family, friendship)
- AI-powered analysis of every entry

#### 2. **Advanced AI Analysis**
- **Pattern Recognition**: Detects 15+ types of red flags including:
  - Gaslighting
  - Isolation tactics
  - Love bombing
  - Financial control
  - Threats and intimidation
  - Boundary violations
  - Workplace bullying
  - Family manipulation
  - And more...
- **Sentiment Analysis**: Tracks emotional tone over time
- **Severity Assessment**: Categorizes concerns as low, moderate, high, or critical
- **Contextual Suggestions**: Provides actionable advice based on detected patterns

#### 3. **Rich Visualizations**
- **Mood Ring**: Interactive animated visualization of overall emotional health (0-100 score)
- **Trend Charts**: Line graphs showing mood progression over time
- **Radar Chart**: Multi-dimensional relationship health assessment across 6 key areas:
  - Safety
  - Trust
  - Respect
  - Independence
  - Communication
  - Boundaries
- **Heat Map Calendar**: Monthly view showing activity intensity and emotional patterns
- **Red Flag Bar Chart**: Frequency analysis of detected warning signs

#### 4. **Comprehensive Education Library**
- 15+ detailed red flag definitions
- Real-world examples for each pattern
- "What To Do" guidance
- Resource links and hotlines
- Categorized by:
  - Emotional Manipulation
  - Control & Isolation
  - Safety Concerns
  - Boundaries & Respect
  - Workplace Issues
  - Family Dynamics

#### 5. **AI Counselor Chat**
- 24/7 supportive AI chatbot
- Context-aware responses based on journal history
- Empathetic, non-judgmental guidance
- Crisis detection and resource recommendations
- Secure, private conversations

#### 6. **Privacy & Security**
- **PIN Lock**: Biometric-style authentication
- **Auto-Lock**: Locks when app is backgrounded
- **Disguised Mode**: 
  - Instant panic exit (Shift+Esc)
  - Disguised app icons (Calculator, Notes, Calendar)
  - Quick-exit to fake interface
- **Local-First**: All data stored on device
- **Encrypted Storage**: Sensitive data encrypted
- **No Tracking**: Zero data collection
- **Offline Mode**: Full functionality without internet

### Additional Features

- **Data Export**: Backup all journal entries and insights as JSON
- **Streak Tracking**: Monitor journaling consistency
- **Daily Reminders**: Gentle prompts for self-reflection (optional)

## üì± Usage Guide

### Getting Started

1. **Onboarding**: Set your PIN and choose your primary focus area
2. **First Entry**: Write about a recent experience

4. **Track Progress**: Use Insights tab to see trends over time
5. **Learn**: Browse the Library to understand red flags
6. **Get Support**: Chat with AI counselor when needed

### Key Interactions

- **New Entry**: Tap the "+" button on Home or Journal tab
- **View Insights**: Check Insights tab for visualizations
- **Search Library**: Use search bar to find specific red flags
- **Lock App**: Settings ‚Üí Lock App Now
- **Emergency Exit**: Press Shift+Esc anytime
- **Export Data**: Settings ‚Üí Export All Data

### Safety Tips

- Use disguised icon if someone might access your device
- Enable PIN lock immediately after setup
- Practice the quick-exit gesture (Shift+Esc)
- Export backups regularly to a secure location
- Clear chat history after sensitive conversations
- Trust your instincts - if something feels wrong, it probably is
